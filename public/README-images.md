# Image Notes

Real homestay photos are stored directly inside the `public` folder.

Website image paths currently use root public URLs such as `/nonamanis-luar-1.webp`.

If the project is cleaned up later, the recommended future organization is to move photos into `public/images/` and update references together in one pass.

Recommended image preparation:

- Width around 1200px to 1600px
- WebP preferred
- Compress before upload
- Avoid very large images to keep the website fast
